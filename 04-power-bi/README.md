# 04 — Power BI

Dashboards desenvolvidos em **Power BI** para transformar dados em informação visual e apoiar a tomada de decisão. Complementei o aprendizado das disciplinas com o **Intensivão de Power BI da Hashtag Treinamentos** (8h).

📜 **Certificado:** [`certificado-power-bi-hashtag.pdf`](./certificado-power-bi-hashtag.pdf)

## 🛠️ Habilidades aplicadas

- Importação e modelagem de dados (várias tabelas)
- Criação de visuais (gráficos de barras, comparativos, indicadores)
- Comparação de cenários *antes x depois*
- Storytelling de dados

## 📁 Dashboards

### 👵 População Idosa no Brasil — Projeto de Extensão
Dashboard que analisa a **distribuição da população feminina 60+** por região do Brasil e por **tipo de território** (área urbana, área rural, favelas e comunidades urbanas, terras indígenas), além de indicadores de infraestrutura. Construído a partir de dados reais do **Censo 2022 (IBGE)** e da **PNAD**, no contexto de um projeto de extensão com impacto social.

- [`populacao-idosa-projeto-extensao.pbix`](./populacao-idosa-projeto-extensao.pbix)

### 🐧 Imputação de Dados — Palmer Penguins
Dashboard que apresenta o resultado do tratamento de dados do trabalho de Mineração de Dados, comparando a base **antes e depois da imputação**: contagem de valores ausentes (NaN) por coluna e distribuição das variáveis (`bill_length`, `bill_depth`, `body_mass`, `flipper`) antes e depois.

- [`penguins-imputacao.pbix`](./penguins-imputacao.pbix)
- 🔗 Relacionado ao notebook em [`03-mineracao-de-dados`](../03-mineracao-de-dados)

> 💡 Para abrir os arquivos `.pbix`, é necessário ter o **Power BI Desktop** (gratuito) instalado.
